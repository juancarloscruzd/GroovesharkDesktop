Grooveshark Desktop Download

This is an unofficial desktop client for Grooveshark.com. It features:

  • Bowtie-like miniController with fully customizable themes
  • Media key support
  • Growl notifications
  • reduced pixel waste
  • native Last.fm support
  • miniController, minify, and fullscreen view modes

Installer Package Disk Image

I recommend that you install using the installer package. If you are wary of installing from a package, a disk image is provided for your convenience.

coming soon

  • More themes(?)
  • Consolidated user interface for settings like theme repo URL and default theme
  • Real miniController theme creation documentation. In the meantime, you can look at this directory to get a sense of how simple this is. It's rather similar to Bowtie.
    • theme.js - your main theme file. Contains all the required functions and essential data (like width and height)
    • theme.html - the HTML structure of your theme. Please contain everything within a div with id="microController".
    • theme.css - your stylesheet will automatically be added when your theme is applied, and removed when expanded
    • You need to include some sort of preview and choose a unique shortname for your theme.

Notes

Some visual resources will be retrieved from jake.teton-landis.org

As Grooveshark will change their application code without notice, I'm abstracting some important functionality, like player control, to provide a stable API for possible remote authors, if I ever get themes off the ground.

If you want to hack on this script, I'd recommend downloading a copy of Grooveshark's app.js file. You can use Opera 9.27 and http://www.howtocreate.co.uk/tutorials/jsexamples/JSTidy.html to tidy up the code.

Unfortunately, core.js does not run in Opera 9.27 so that's too hard to deal with. As such, all Grooveshark features dealt with in core.js will remain (mostly) untampered with.
